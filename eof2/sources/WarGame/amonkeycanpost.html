== Introduction == Greasemonkey is a nice extension for Firefox that let you to "mod" the way web pages look. A greasemonkey script is written in simple javascript using only some special comments at the beginning of it. For more info visit the official site: http://www.greasespot.net/ == The w0rm == I wrote a simple worm to show how this extension can be abused, all in few lines of js.
